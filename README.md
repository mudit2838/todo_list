# Todo List App

A simple Todo List Web Application built with Node.js, Express, MongoDB Atlas, and EJS.  
It allows users to create, edit, delete, and set priorities for tasks.

## Live Demo

Deployed on Render: [Todo List Live Demo on Render](https://todo-list-5bbi.onrender.com)

## Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Set priority levels: Low, Medium, High  
- Stores data in MongoDB Atlas  
- Uses EJS templates for rendering  

## Tech Stack

- **Backend:** Node.js, Express  
- **Frontend:** EJS, HTML, CSS  
- **Database:** MongoDB Atlas  
- **Deployment:** Render  
- **Version Control:** Git and GitHub
